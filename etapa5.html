Etapa 5 - Ativar o GitHub Pages

  1. No site do GitHub, abre o repositório (meu-site-github).
  2. Vai a Settings → Pages. Em Source, escolhe:
    • Deploy from a branch
    • Branch: main
    • Folder: / (root)
  3. Clica em Save. Aguarda alguns segundos e o GitHub mostrará o link do site:
    4. https://.github.io/meu-site-github/
  4. (Exemplo: https://joaogouveia-programador.github.io/meu-site-github/)
