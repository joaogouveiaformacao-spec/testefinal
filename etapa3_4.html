Etapa 3 - Criar o ficheiro do Sítio

Na pasta local do repositório, cria um novo ficheiro com o nome: index.html
  1. Abre-o com o Notepad++.

  3. Guarda o ficheiro e volta ao GitHub Desktop.

Etapa 4 - Fazer o 1º Commit e Push

  1. No GitHub Desktop, verifica que o ficheiro index.html aparece na lista de alterações.
  2. Escreve uma mensagem de commit, por exemplo:
    3. Adicionado ficheiro index.html
  3. Clica em Commit to main.
  4. Depois, clica em Push origin para enviar as alterações para o GitHub.
